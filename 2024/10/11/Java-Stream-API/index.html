🚀 Getting Started with Java Stream API

🥇 Java Stream API, introduced in Java 8, is a powerful tool for processing collections in a functional programming style. It allows you to perform operations like filtering, mapping, and reducing in a more readable and efficient way. You can get rid of tedious for/while loops and just write code like a neat pipeline!

📌 What is a Stream?

  • A Stream is a sequence of elements from a source (like a collection) that supports aggregate operations.
  • Streams do not store data; they operate on the original data source and genenrally create a new one.
  • Operations are divided into:
    • Intermediate operations: Return a new stream (e.g., filter(), map()).
    • Terminal operations: Produce a result or a side effect (e.g., collect(), forEach()).

🛠️ Creating a Stream

For an array x, you can use Arrays.toStream(x) to get a stream object, while for an List x, you can use x.stream()

📖 Common Used Stream Operations

1. filter()

Filters elements based on a given predicate (condition).

2. peek()

Performs an action for each element without creating a new one. This is helpful for debugging or modifying the attributes.

3. map()

Transforms each element of the stream into another object or value.

4. collect()

Collects elements from the stream into a collection (e.g., List, Set).
